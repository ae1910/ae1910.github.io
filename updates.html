Final Site: The Top Week 4 - 6

With what I learned about javascript, I felt that I could now make a site that incorporates what I already knew and what I recently learned.

For the site, I decided I wanted to create a news website called topnews that will display news from various news sources. The main focus when creating the site will be its design and the use of an API. The API that will be used is from News API. This returns live news, headlines, and blog articles from across the web.

Creating the Proof of Concept

The first thing I did was create a wireframe of the site to get a basic idea of how the pages would look like. I used Adobe XD to create the wireframe.

After several tweaks, this was the result of some of the pages:

The main pages will be the homepage, the pages for the articles, and the search results page. As well, I plan to make the design mobile friendly.

When you first open the site, you are open to a slider that shows the thumbnail of the 4 latest news from the API. As you scroll, you will see the 4 most recent articles for each category of news articles. This can be sports, business or entertainment. At the end of the page will be a footer, that will include the option to "sign up" for the newsletter of the site that would keep the user updated to the latest news.

The next pages will be for each category of news. In these pages it will be a list of the articles for whichever category is chosen.

The last page I plan to also do is a search results page that will show results for what you want to search for.

When an article is chosen, you are taken to the source of where the article is from.

The Process

Starting with the slider on the main page, with how the News API is set up, I used the top headlines endpoint. This endpoint returns breaking news headlines for countries, categories, and singular publishers. For this site, I used the endpoint to return for news headlines only from the US. Also, based on the documentation fo the API, it is perfect for the use of live up-to-date news headlines, which is perfect for what I wanted for the slider.

To set up the slider, I used Javascript, but I did not know how to set it up. For the slider, I wanted it to have left and right buttons, as well has so the user can manually click through the slides of the recent news. As well, I wanted the slides to automatically change slides. So, I followed a YouTube video to help design it. Then like what I did, when I was following the example in MDN's Working with JSON data article, I fetched the API and used the JSON data tha was returned to fill the slider with the 4 most recent news.

I did the same thing for the sections of headlines that are split into categories that follows the slider. As well as for the pages of headlines of each category. There are 7 categories: general, business, entertainment, health, science, sports, and technology.

Finally, I did the search results page. Instead of using the top headlines endpoint, I used their other main endpoint; the everything endpoint. This allows to search every article published by 80,000 different sources in the last 4 years.

I added search bars on each page, so that whatever the user inputs, it will be used to search for in the article title and body. Then, when the input is submitted, the user will be taken to the search results page showing all the articles that mention what the user searched for. However, the results won't be shown all at once, only the 10 first ones will show, then the user has the option to show the next 10 by clicking the "Show More" options. The user can continue to do so until all the results are shown. allows automatically, the results are sorted by relevancy, but it can also be ordered by popularity and be recentness.
